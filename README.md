# vim

Notes for vim operation

Split windows

```
:sp
:vs   # for veretical split
```

Buffer

```
:badd <filename>
:ls  # show buffers
:b<index>  # switch buffer
```

## Plugins

Run Prettier

```
:Prettier
```

Fold code

```
za  # just use za in the edit mode
```

Show folder tree panel

```
:NERDTree

# To switch between file and folder tree
control + w
```
